The Sudoku Game is a logic-based number puzzle where users must fill a 9√ó9 grid according to standard Sudoku rules.

- The grid consists of 81 cells divided into nine 3√ó3 subgrids.
- Players must fill empty cells with numbers from 1 to 9.
- Each number can only appear once per row, once per column, and once per 3√ó3 subgrid.
- The game is completed when the entire grid is correctly filled according to Sudoku rules.

This project is implemented in Java, using a GUI framework for the user interface.
